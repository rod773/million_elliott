## 8.2 Example Trade 1 — Setup A: Wave 5 Sell

### Context:
AUD/USD has been rallying strongly from 0.6200 for several months.
The Weekly chart shows a clear 5-wave impulse structure forming.

---

### Step 1: Weekly Chart Analysis
```
Wave 1: 0.6200 → 0.6550 (+350 pips)
Wave 2: 0.6550 → 0.6350 (-200 pips, 57% retrace — valid)
Wave 3: 0.6350 → 0.6980 (+630 pips, 1.8× Wave 1 — Wave 3 is longest ✅)
Wave 4: 0.6980 → 0.6820 (-160 pips, 25% retrace — shallow, alternation with deeper W2 ✅)
Wave 5: 0.6820 → ??? (projected target)
```

**Fibonacci target for Wave 5:**
Wave 1 length = 350 pips
100% extension from Wave 4 start: 0.6820 + 0.0350 = 0.7170
61.8% extension from Wave 4 start: 0.6820 + 0.0216 = 0.7036

**Watch zone: 0.7036 – 0.7170**

---

### Step 2: Daily Chart Analysis
```
Wave 5 sub-structure forming:
  5-i: 0.6820 → 0.6920
  5-ii: 0.6920 → 0.6860 (retrace ✅)
  5-iii: 0.6860 → 0.7050 (strongest sub-wave ✅)
  5-iv: 0.7050 → 0.6980 (consolidation ✅)
  5-v: 0.6980 → 0.7080? (final push)
```

Wave 5-v is approaching the 61.8% extension zone (0.7036–0.7080).

---

### Step 3: 4H Chart — MACD Divergence Check
```
Price Peak 1: 0.7050 (during wave 5-iii)
  → MACD Histogram: +0.0048

Price Peak 2: 0.7085 (during wave 5-v) — HIGHER PRICE
  → MACD Histogram: +0.0029 — LOWER MACD

BEARISH DIVERGENCE CONFIRMED ✅
```

MACD signal line has crossed below MACD line on the 4H chart — confirmation.

---

### Step 4: 1H Chart — Entry Timing
```
At 0.7085: Bearish pin bar candle closes (long upper wick, small body)
Next candle opens: 0.7080

Entry signal: Bearish pin bar close = SELL
```

---

### Step 5: Trade Execution
```
SETUP A — WAVE 5 SELL

Entry:       0.7078 (at bearish pin bar close)
Stop Loss:   0.7120 (above Wave 5 high of 0.7085 + 35-pip buffer = 42 pips risk)

Target 1:    0.6820 (Wave 4 territory = Wave A target, 258 pips)  R:R = 6.1:1
Target 2:    0.6580 (61.8% retrace of entire impulse, 498 pips)    R:R = 11.9:1
Target 3:    0.6350 (full retrace to Wave 2 low, 728 pips)          R:R = 17.3:1

Position Sizing (account $50,000, 1% risk = $500):
  Position size = $500 / (42 pips × $10) = 1.19 lots ≈ 1.2 lots

Partial exits:
  At T1 (0.6820): Close 0.5 lots → +$1,290 profit, move stop to breakeven
  At T2 (0.6580): Close 0.5 lots → +$2,490 additional profit
  At T3 (0.6350): Close 0.2 lots → +$1,456 additional profit

Total trade profit: ~$5,236 on $500 risk
Net R made: ~10.5R
```

---

## 8.3 Example Trade 2 — Setup B: Wave C Bottom Buy

### Context:
AUD/USD completed a 5-wave decline from 0.7800 to 0.6150 over 18 months.
An A-B-C corrective rally has been underway. Looking for the end of Wave C
to short the continuation of the downtrend.

Wait — this is actually a **bearish** continuation trade (short Setup B).

---

### Step 1: Weekly Chart Analysis
```
Completed 5-wave decline:
  Wave 1 down: 0.7800 → 0.7200 (-600 pips)
  Wave 2 up: 0.7200 → 0.7500 (+300 pips, 50% retrace)
  Wave 3 down: 0.7500 → 0.6500 (-1000 pips, largest wave ✅)
  Wave 4 up: 0.6500 → 0.6800 (+300 pips, 30% retrace ✅ alternation with W2)
  Wave 5 down: 0.6800 → 0.6150 (-650 pips)

A-B-C corrective rally now underway:
  Wave A: 0.6150 → 0.6600 (+450 pips)
  Wave B: 0.6600 → 0.6350 (-250 pips, 56% retrace of A ✅)
  Wave C: 0.6350 → ??? (target = Wave A length = +450 pips → 0.6800)
```

**Wave C target:** 0.6350 + 0.0450 = 0.6800
Also: 61.8% retracement of full 5-wave decline = 0.6150 + (0.7800-0.6150) × 0.618 = 0.7170

**Watch zone for Wave C top: 0.6750–0.6850**

---

### Step 2: Daily Chart Analysis
```
Wave C sub-structure:
  C-1: 0.6350 → 0.6550 (+200 pips)
  C-2: 0.6550 → 0.6450 (-100 pips)
  C-3: 0.6450 → 0.6750 (+300 pips)
  C-4: 0.6750 → 0.6680 (-70 pips)
  C-5: 0.6680 → 0.6810? (approaching target)

Wave C completing at ~0.6810 (resistance zone, previous Wave 4 territory).

### Step 3: 4H Chart — MACD Divergence

### Step 4: 1H Chart — Entry

```

### Step 5: Trade Execution

## 8.5 The Setup A and B Pattern Recognition Guide

### Quick-Reference Conditions:

**Setup A: Sell at Wave 5 top**
```
✓ 5 clear waves up on Daily/4H
✓ Wave 5 at Fibonacci target (0.618–1.0 × Wave 1 extension)
✓ MACD bearish divergence on 4H (price: higher high, MACD: lower high)
✓ Reversal candle (pin bar or engulfing)
✓ Previous resistance level nearby
→ SELL with stop above Wave 5 high
→ Targets: Wave A (38.2%), Wave C (61.8%) of impulse
```

**Setup A: Buy at Wave 5 bottom (inverse)**
```
✓ 5 clear waves DOWN on Daily/4H
✓ Wave 5 at Fibonacci target below Wave 3 low
✓ MACD bullish divergence on 4H (price: lower low, MACD: higher low)
✓ Reversal candle
✓ Previous support level nearby
→ BUY with stop below Wave 5 low
→ Targets: Wave A (38.2%), Wave C (61.8%) rally
```

**Setup B: Short Wave C completion (trend continuation sell)**
```
✓ Prior 5-wave decline completed
✓ A-B-C counter-trend rally underway
✓ Wave C = Wave A in length, at Fibonacci resistance (50–61.8%)
✓ MACD bearish divergence at Wave C top
✓ Reversal candle
→ SELL: targets back to and below Wave 5 low
```

**Setup B: Long Wave C completion (trend continuation buy)**
```
✓ Prior 5-wave rally completed
✓ A-B-C correction underway
✓ Wave C = Wave A, at Fibonacci support (50–61.8% of rally)
✓ MACD bullish divergence at Wave C low
✓ Reversal candle
→ BUY: targets new all-time high for the instrument
```

## 8.6 Building Your Own Watchlist Process

### Weekly (every Sunday):
1. Open AUD/USD Weekly chart
2. Update wave count — where are we?
3. Identify the next high-probability zone (Wave 5 target? Wave C target?)
4. Set price alerts at those levels
5. Note which setup is expected when the alert triggers

### Daily (every morning):
1. Check if any alerts were triggered
2. Open 4H chart — is MACD divergence forming?
3. If yes: move to 1H for entry timing
4. If no: set a deeper alert and go back to watching

### This process takes 15–20 minutes per day. Nothing more is needed.
