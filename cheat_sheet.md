# 📋 AUD/USD Elliott Wave + MACD Divergence — Quick Reference Cheat Sheet

---

## ELLIOTT WAVE RULES (Hard — Never Break)
| Rule | What It Means |
|------|--------------|
| Wave 2 ≤ 100% of Wave 1 | Wave 2 never goes below Wave 1 start |
| Wave 3 ≠ shortest | Wave 3 must be longer than W1 or W5 |
| Wave 4 ≠ overlap Wave 1 | Wave 4 cannot enter Wave 1 price territory |

## KEY FIBONACCI LEVELS
| Level | Used For |
|-------|---------|
| 38.2% | Wave 4 retracement; T1 target |
| 50.0% | Wave 2 retracement |
| 61.8% | Wave 2 deep retrace; Wave C target; T2 target |
| 100% | Wave C = Wave A; Wave 5 = Wave 1 |
| 161.8% | Wave 3 extension target |

## MACD DIVERGENCE RULES
| Type | Price | MACD | Signal |
|------|-------|------|--------|
| Regular Bearish | Higher High | Lower High | SELL |
| Regular Bullish | Lower Low | Higher Low | BUY |
| Hidden Bullish | Higher Low | Lower Low | BUY (continuation) |
| Hidden Bearish | Lower High | Higher High | SELL (continuation) |

## TWO SETUPS
```
SETUP A: Wave 5 Reversal
→ 5-wave impulse complete
→ MACD regular divergence at Wave 5
→ Entry: reversal candle
→ Stop: beyond Wave 5
→ Targets: A (38.2%), C (61.8%), full retrace

SETUP B: Wave C / Wave 2 Entry
→ A-B-C correction complete (or Wave 2 of new impulse)
→ MACD regular divergence at Wave C/2
→ Entry: reversal candle
→ Stop: beyond Wave C/2
→ Targets: new Wave 3 (161.8% extension)
```

## POSITION SIZING


## HARD LIMITS
- Daily max loss: 3% → STOP for the day
- Monthly max drawdown: 8% → STOP, review
- Consecutive losses: 3 → STOP for the day
- Minimum R:R: 1:2 (prefer 1:3+)
- No trading 2H before/after high-impact news

## 5-STEP ANALYSIS PROCESS
1. Weekly chart → Establish wave count & trend
2. Daily chart → Identify current wave position
3. 4H chart → Look for MACD divergence
4. 1H chart → Time the entry candle
5. Execute → Pre-defined stop, targets, position size

## CONFLUENCE MINIMUM: 7 out of 12 factors
Wave count clear | At Fibonacci level | MACD divergence |
Signal line crossed | Spans 20+ candles | Two clear peaks |
Reversal candle | S/R level | No news in 4H |
Higher TF supports | Alternation respected | Macro aligns |
