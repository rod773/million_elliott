# Module 02 — Elliott Wave Theory Mastery

---

## 2.1 The Core Concept

Elliott Wave Theory, developed by Ralph Nelson Elliott in the 1930s, states that financial
markets move in **repetitive, fractal wave patterns** driven by crowd psychology.

There are two types of waves:
- **Impulse waves** — move IN the direction of the trend (5 waves)
- **Corrective waves** — move AGAINST the trend (3 waves, or complex variations)

The complete cycle = **5 impulse waves + 3 corrective waves = 8 waves total**

---

## 2.2 The 5-Wave Impulse Structure

```
        3
       / \
      /   \
     1     \    5
    / \     \  / \
   /   \     \/   \
  /     2    4     \
 /                  \
Start              End
```

| Wave | Character | Description |
|------|-----------|-------------|
| Wave 1 | Weak | Few notice the new trend; often mistaken for a pullback |
| Wave 2 | Retracement | Deep pullback (38.2%–61.8% of Wave 1), shakes out weak hands |
| Wave 3 | Strongest | Most powerful move; where most money is made |
| Wave 4 | Pullback | Shallow retracement (38.2% of Wave 3); more complex |
| Wave 5 | Final push | Often shows MACD divergence — KEY SIGNAL |

---

## 2.3 The 3-Wave Corrective Structure (A-B-C)

After the 5-wave impulse, price corrects in an A-B-C pattern:

```
     B
    / \
   /   \
  A     \
          C
```

| Wave | Description |
|------|-------------|
| Wave A | First leg of correction — often mistaken for Wave 4 continuation |
| Wave B | Partial retracement of A — creates false hope |
| Wave C | The killer drop/rally — completes correction, often equals A in length |

---

## 2.4 Key Elliott Wave Rules (NEVER Break These)

These are hard rules. If your count violates them, the count is WRONG.

### Rule 1: Wave 2 cannot retrace more than 100% of Wave 1
- Wave 2 never goes below the start of Wave 1 (in a bull move)

### Rule 2: Wave 3 is never the shortest impulse wave
- Wave 3 is almost always the longest
- At minimum, Wave 3 must be longer than either Wave 1 or Wave 5

### Rule 3: Wave 4 cannot overlap Wave 1 territory
- The end of Wave 4 cannot enter the price territory of Wave 1
- **Exception:** Diagonal triangles only

---

## 2.5 Key Elliott Wave Guidelines (Strong Tendencies)

These are not rules but are highly reliable:

| Guideline | Description |
|-----------|-------------|
| Alternation | If Wave 2 is sharp, Wave 4 is flat (and vice versa) |
| Wave 3 extension | Wave 3 typically extends to 1.618 × Wave 1 |
| Wave 5 equality | Wave 5 often equals Wave 1 in length |
| Wave 2 depth | Usually retraces 50%–61.8% of Wave 1 |
| Wave 4 depth | Usually retraces 38.2% of Wave 3 |

---

## 2.6 Fibonacci Levels in Elliott Wave

Fibonacci is the mathematical backbone of Elliott Wave. These are the key levels:

### For retracements:
- **38.2%** — Common Wave 4 retracement
- **50.0%** — Common Wave 2 retracement
- **61.8%** — Deep Wave 2 retracement (Golden Ratio)

### For extensions (price targets):
- **100%** — Wave C = Wave A; Wave 5 = Wave 1
- **127.2%** — Wave 3 minimum extension
- **161.8%** — Most common Wave 3 extension (Golden Ratio extension)
- **261.8%** — Extended Wave 3 target

> 💡 **How to use:** Draw Fibonacci from the START to END of Wave 1.
> The retracement levels give you Wave 2 entry zones.
> The extension levels give you Wave 3 and Wave 5 targets.

## 2.7 Corrective Wave Patterns

Beyond simple A-B-C, corrections can take several forms:

### Zigzag (5-3-5)
```
A = 5-wave move down
B = 3-wave move up (retraces 38.2%–78.6% of A)
C = 5-wave move down (usually equals A)
```

### Flat (3-3-5)
```
A = 3-wave move down
B = 3-wave move up (retraces close to 100% of A)
C = 5-wave move down (usually equals or exceeds A)
```

### Triangle (3-3-3-3-3)
```
Five overlapping waves labeled A-B-C-D-E
Converging trendlines
Usually appears as Wave 4 or Wave B
Followed by a "thrust" equal to the widest part of the triangle
```

### Complex Corrections (Double/Triple Threes)
```
W - X - Y patterns
W and Y = corrective patterns
X = connecting wave
Most common when market is in a larger degree sideways correction
```

## 2.8 Elliott Wave Degrees (Timeframe Hierarchy)

Elliott Wave is fractal — the same patterns repeat at every degree:

| Degree | Timeframe | Duration |
|--------|-----------|----------|
| Grand Supercycle | Monthly (decades) | 50–100 years |
| Supercycle | Monthly | 10–50 years |
| Cycle | Weekly/Monthly | 1–10 years |
| Primary | Weekly | Months to years |
| Intermediate | Daily | Weeks to months |
| Minor | 4-Hour/Daily | Days to weeks |
| Minute | 1-Hour | Hours to days |
| Minuette | 15-Min | Minutes to hours |

**For this course, we focus on:**
- **Primary / Intermediate** (Weekly/Daily) for wave count and bias
- **Minor / Minute** (4H / 1H) for entries

## 2.9 Identifying Wave 3 — Where the Money Is Made

Wave 3 is the money wave. Here's how to identify it:

**Characteristics of Wave 3:**
- Strongest momentum (MACD histogram peaks here)
- Highest volume
- Breaks above Wave 1 high decisively
- Moves fast — gaps are common
- News often aligns with the direction

**Entry strategy for Wave 3:**
1. Identify completed Wave 1 and Wave 2
2. Wave 2 must pull back to 38.2%–61.8% Fibonacci of Wave 1
3. Look for MACD bullish signal at Wave 2 low (see Module 03)
4. Enter at Wave 2 completion, target 1.618 × Wave 1 for Wave 3

---

## 2.10 Identifying Wave 5 — The MACD Divergence Opportunity

Wave 5 is critical because it almost always produces **MACD divergence**.

**Characteristics of Wave 5:**
- Price makes new high (above Wave 3 high)
- BUT momentum is weaker than Wave 3
- MACD histogram makes a lower high while price makes a higher high
- This is **bearish divergence** — major sell signal
- Wave 5 often targets 61.8%–100% of Wave 1 length

**This is one of the two core trade setups in this course.**

## 2.11 Practical Wave Counting on AUD/USD — Step by Step

### Step 1: Start on the Weekly Chart
- Find the most obvious major swing high and low in the last 2–5 years
- Label the highest point and lowest point
- Determine if the dominant trend is UP or DOWN

### Step 2: Count the 5-Wave Structure
- In an uptrend: 5 waves up, then A-B-C correction down
- In a downtrend: 5 waves down, then A-B-C correction up
- Use Fibonacci to validate your count

### Step 3: Drop to the Daily Chart
- Refine the wave count within each major wave
- Each wave on the Weekly should sub-divide on the Daily

### Step 4: Drop to the 4H Chart
- This is where you look for your entries
- Use MACD divergence to time entries at wave completion points

## 2.12 Common Elliott Wave Mistakes to Avoid

| Mistake | Why It's Wrong | Fix |
|---------|----------------|-----|
| Forcing a count | Making price fit your bias | Let price tell you the wave |
| Ignoring rules | Accepting a Wave 4 that overlaps Wave 1 | Recount entirely |
| Single-scenario thinking | Only having one wave count | Always have an alternate count |
| Skipping Fibonacci | Counting without levels | Always draw Fibonacci |
| Counting on 1 timeframe | No top-down confirmation | Always check higher TF |

## 2.13 Module 02 Summary

- Markets move in 5-wave impulses and 3-wave corrections
- Three hard rules: Wave 2 < 100% of Wave 1; Wave 3 never shortest; Wave 4 no overlap
- Fibonacci levels validate wave counts and provide targets
- Wave 3 = strongest momentum = best entry after Wave 2 retracement
- Wave 5 = divergence signal = great reversal trade entry
- Always use top-down analysis across timeframes
